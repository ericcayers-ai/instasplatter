# DESIGN.md — InstaSplatter shell

## Direction
Midnight capture bay. Teal pigment is the brand; warm brass marks live progress. Surfaces stay quiet so the viewport is the stage.

## Tokens
- Accent / brand: teal `#2fc4b0` (light: `#0e7a6d`)
- Progress / live: brass `#e0a34a` (replaces purple Trace)
- Neutrals: cool ink glass `#0a0c10` / `#11151c` / `#171c26`
- Light theme: near-pure `#f5f6f8` surfaces (not cream)

## Navigation QOL
- Suite segmented control (Recon / Geo)
- Export consolidated into one menu
- Side panel icon toggle; Settings with `Ctrl+,`
- Shortcuts: `Ctrl+B` panel, `Ctrl+L` log, `Esc` dismiss overlays
- Status pills for Ready / Live / Complete / Failed
- Home: brand-first hero, full-bleed drop plane, recent projects inline

## Motion
150–220ms ease-out. Loading uses dual pulse rings + shimmer progress (transform scaleX). Reduced-motion honored.
